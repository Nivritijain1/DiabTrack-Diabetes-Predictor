# DiabTrack-Diabetes-Predictor
A clinical system that predicts diabetes through ensemble ML, integrating NHANES population data with PIMA Indian diabetes records, harmonizing 10,000+ patient profiles. Core architecture employs a weighted voting ensemble: Random Forest (400 trees) paired with 2 XGBoost variants (300/200 trees) at 1.3:1.0:0.7 ratios, Achieving 90.2% accuracy
